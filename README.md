# 🚀 DDoS Testing Tool (For Educational & Cybersecurity Purposes Only)

## 📌 Description
This project is a **Python-based network stress testing tool** designed for **educational purposes** and **authorized security testing**.

It helps developers and cybersecurity learners understand:
- How high traffic affects servers  
- Basic concepts of network stress testing  
- Performance limitations of web services

## 🎯 Features
- Multi-threaded request system  
- Custom target support (IP / Domain)  
- Adjustable thread count  
- Simple CLI interface  
- Lightweight and fast execution  

---

## 🛠️ Installation

```bash
git clone https://github.com/kritanshshukla373-afk/Kabir-ddos
cd Kabir-ddos
```

---

## ▶️ Usage

```bash
python3 kabirXshaurya.py -s <target> -p <port> -t <threads>
```

### Example:
```bash
python3 kabirXshaurya.py -s example.com -p 80 -t 100
```
